# 📄 Pagination

Implementing pagination techniques in Python.

## Files

| File | Description |
|------|-------------|
| `0-simple_helper_function.py` | Simple pagination helper |
| `1-simple_pagination.py` | Basic pagination |
| `2-hypermedia_pagination.py` | Hypermedia pagination (HATEOAS) |
| `3-hypermedia_del_pagination.py` | Deletion-resilient pagination |

## Key Concepts

- Page/size-based pagination
- Offset/limit pagination
- Hypermedia pagination with metadata
- Handling deleted records in pagination

## Usage

```bash
python3 1-main.py
```
