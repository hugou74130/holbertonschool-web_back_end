# 🔄 Python Async Comprehension

Using async generators and comprehensions in Python.

## Files

| File | Description |
|------|-------------|
| `0-async_generator.py` | Async generator yielding random numbers |
| `1-async_comprehension.py` | Async list comprehension |
| `2-measure_runtime.py` | Measure parallel execution time |

## Key Concepts

- `async def` generators
- `async for` loops
- Async comprehensions
- `asyncio.gather()` for parallelism

## Usage

```bash
python3 0-main.py
```
