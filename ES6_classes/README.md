# ES6_classes
Ce dossier contient des exercices pour pratiquer les fonctionnalités des classes en ES6.

## Exercices
* 0-classroom.js : définition d'une classe
* 1-make_classrooms.js : création d'instances de classes
* 2-hbtn_course.js : utilisation des constructeurs et des propriétés
* 3-currency.js : utilisation des getters et des setters
* 4-pricing.js : utilisation des méthodes statiques
* 5-building.js : utilisation des classes abstraites
* 6-sky_high.js : utilisation de l'héritage
* 7-airport.js : utilisation des représentations de chaîne personnalisées
* 8-hbtn_class.js : utilisation des converters
* 9-hoisting.js : utilisation de l'ordre de création des classes et des instances
* 10-car.js : utilisation des méthodes pour cloner des objets

## Tests
Les tests pour les exercices sont fournis séparément. Pour éxécuter les tests, il faut utiliser un framework de test JavaScript tel que Jest ou Mocha.
