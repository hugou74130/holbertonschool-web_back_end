## 🧐 About <a name = "about"></a>

This repository contains the web back-end specialization projects from Holberton School France. It covers modern JavaScript (ES6), Python asynchronous programming, NoSQL databases with MongoDB, and efficient data pagination techniques.

## 🏁 Getting Started <a name = "getting_started"></a>

### Prerequisites

- Node.js 14+
- Python 3.8+
- MongoDB (for NoSQL modules)

### Installing

Clone the repository:

```bash
git clone https://github.com/hugou74130/holbertonschool-web_back_end.git
cd holbertonschool-web_back_end
```

## 📦 Modules <a name = "modules"></a>

| Module | Description | Language |
|--------|-------------|----------|
| `ES6_basic/` | ES6 fundamentals — arrow functions, let/const, template literals | JavaScript |
| `ES6_classes/` | Object-oriented programming with ES6 classes | JavaScript |
| `ES6_data_manipulation/` | Map, Filter, Reduce, Typed Arrays, Sets | JavaScript |
| `ES6_promise/` | Promises, async/await, error handling | JavaScript |
| `NoSQL/` | MongoDB basics — CRUD operations with pymongo | Python |
| `pagination/` | Simple, hypermedia, and deletion-resilient pagination | Python |
| `python_async_comprehension/` | Async generators and comprehensions | Python |
| `python_async_function/` | Asyncio basics — coroutines, tasks, runtime measurement | Python |
| `python_variable_annotations/` | Type hints and variable annotations (mypy) | Python |

## 🎈 Usage <a name = "usage"></a>

Run individual Python or JavaScript files:

```bash
# Python example
python3 python_async_function/0-basic_async_syntax.py

# Node.js example
node ES6_basic/0-constants.js
```
