# 🍃 NoSQL with MongoDB

Introduction to NoSQL databases using MongoDB and Python.

## Files

| File | Description |
|------|-------------|
| `0-list_databases` | List all MongoDB databases |
| `1-use_or_create_database` | Create/use a database |
| `2-insert` | Insert a document |
| `3-all` | List all documents |
| `4-match` | Match documents |
| `5-count` | Count documents |
| `6-update` | Update documents |
| `7-delete` | Delete documents |
| `8-all.py` | Python: list all documents |
| `9-insert_school.py` | Python: insert into collection |
| `10-update_topics.py` | Python: update topics |
| `11-schools_by_topic.py` | Python: find by topic |
| `12-log_stats.py` | Python: Nginx log statistics |

## Prerequisites

- MongoDB installed and running
- Python 3.8+
- `pymongo` package

## Usage

```bash
python3 8-all.py
```
